# Programs Implemented

## 1. FizzBuzz

Prints:
- `Fizz` for multiples of 3
- `Buzz` for multiples of 5
- `FizzBuzz` for multiples of both
- otherwise prints number

### Run

```bash
cd FizzBuzz
dotnet run
```
